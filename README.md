# Rowfy
Dynamically add/remove table row using jquery

## Usage

>Html

In table there must be a `tbody` tag & table must contain class `rowfy`.
```
  <table class="table table-striped table-hover rowfy">
    <thead>
    </thead>
    <tbody>
    </tbody>
  </table>
  ```
  
  >Add Script from rowfy.js
